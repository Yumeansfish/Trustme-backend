# aw-core

Shared Python core libraries for the Trust-me backend workspace.

Modules in this package include:

- `aw_core` for common datatypes, configuration helpers, and logging
- `aw_datastore` for Python-side storage helpers
- `aw_transform` for event transforms used in queries
- `aw_query` for the query language implementation

This package is maintained as part of the monorepo. For supported build and
release instructions, use the top-level backend documentation instead of
treating `aw-core` as a standalone distribution.
